## Commands

```
git init
git add <filename>
git commit -m 'Commit Message'
git add -A
git reset --hard
git status
clear
git branch
git checkout -b <new-branch>
git checkout <branch-name>
git remote add <origin> <url>
git push origin <branch-name>
git tag -a <version> -m "message"
git log             # "q" to quit
```
